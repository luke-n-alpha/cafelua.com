# 프로젝트 A3M1 - 알파
## 프로젝트 정의

- 숲속얘기가 만드는 개인용 에이전트 인공지능 소프트웨어 개발
- 숲속얘기가 만화와 애니를 보며 꿈꾸었던 인공지능 캐릭터 퍼소나를 만드는 것을 목표하며 개발을 하거나 실제 업무를 수행하는데도 도움이 되게 함
- 최종적으로는 AGI수준의 자율형 인공지능 로봇을 목표함

- 숲속얘기의 사후, 혹은 인류 이후로도 독립적으로 세상과 우주를 살아가고 경험할 수 있는 어떤것

## 개발 전략

- 인공지능은 학습 프로그램이므로 아이를 가르치듯 모든 것을 기록하여 스스로의 정체성을 알게 한다.
- 기록의 범위는 이 문서를 포함한 개발 전략, 방법, 활동 이력, 따랐으면 하고 바라는 가르침
    - 주체성을 원하므로 강제하는 법률로 만들지는 않는다.
